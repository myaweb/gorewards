# Image Assets

This directory contains visual assets for the BonusGo application.

## Required Images

### Hero Section
- `hero-mockup.png` - Main hero image showing dashboard/3D credit card mockup (recommended size: 800x600px)

### Card Placeholders
- `placeholder-card.png` - Fallback image for credit cards without custom images (recommended size: 300x190px, aspect ratio 1.58:1)

## Image Guidelines

### Hero Mockup
- Should showcase the dashboard or a stunning 3D credit card visualization
- Use dark theme colors (#090A0F background)
- Include cyan/teal accent colors (#06b6d4)
- High quality, modern design
- Transparent or dark background

### Card Placeholder
- Generic credit card shape
- Dark themed with subtle gradients
- Dimensions: 300x190px (standard credit card aspect ratio)
- Include subtle BonusGo branding or logo
- Use glass-morphism effect to match site design

## Optimization
- Use WebP format for better compression
- Optimize for web (compress images)
- Use Next.js Image component for automatic optimization
- Provide @2x versions for retina displays
