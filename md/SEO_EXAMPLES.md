## 🎨 Design Specifications

### OG Image Dimensions
- Width: 1200px
- Height: 630px
- Aspect Ratio: 1.91:1
- Format: PNG
- File Size: ~50-100KB

### Color Palette
- Background: `#090A0F` (Dark)
- Primary: `#06b6d4` (Cyan)
- Secondary: `#0891b2` (Darker Cyan)
- Text: `#ffffff` (White)
- Muted: `#9ca3af` (Gray)

### Typography
- Brand: Bold, 48px
- Title: Bold, 72px
- Subtitle: Light, 32px
- Badge: Semibold, 20px
